## Live website
- https://namyuen.github.io/Portfolio-/

## GitHub repository
- https://github.com/namyuen/Portfolio-

## Pages included
- `index.html` (Home)
- `about.html` (About Me)
- `projects.html` (Projects & Achievements)
- `style.css` (shared styling and responsive layout)
- `images/profile.svg` (avatar graphic)
- optional `images/profile-2.jpg` (profile photo if uploaded)

## GitHub Pages deployment
1. In repo Settings → Pages
2. Branch: `main`, folder `/` (root)
3. Save
4. Wait a few minutes and visit the live URL above.

## Instructions for customizing
1. Update text in `.html` files
2. Replace `images/profile-2.jpg` with a real photo
3. Commit, push, and verify live URL after deployment
